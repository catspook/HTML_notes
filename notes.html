Extra Markup

!DOCTYPES (see line 1 in code) help tell browser which version of html to use for page. Should always be included.

The id tag allows elements to be styled with CSS

The class tag is used to group elements together. Multiple elements of different types (ie, headers and paragraphs) can have same class.

Block level elements will always be on a newline. Examples are:

  • h1, h2, etc
  • p
  • ul
  • li
  • br /
  • hr /

Inline elements will always appear on same line as their neighboring elements. Examples are:

  • a
    • href
  • b
    • (bold)
  • em
    • (emphasis)
  • img

The div tag allows the elements inside the tag to be grouped together. The contents of the div element will be on a newline. This is really useful when combined with an "id" tag and a "class" tag, for use with CSS. Also helps your code stay neater (like, put each form above inside a "div" so it's easier to keep track of).

The span tag is used like "Div", but inline. Most useful for putting a "class" with it to control CSS.

iframes are basically used to embed a seperate window into your page. Like a google maps screen. p. 189

meta tags store information about the website. See the code in lines 2-10ish.
